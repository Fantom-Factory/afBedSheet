Welcome to BedSheet {{{ bedSheetVersion }}}!

Something fresh and clean to lay your web app on!

BedSheet is a Fantom framework for web applications, providing a rich middleware mechanism for the routing and delivery of content over HTTP.

Full API & fandocs are available on the Fantom Pod Repository.

 

To disable this welcome page, contribute a Route in your App Module:

@Contribute { serviceType=Routes# } static Void contributeRoutes(Configuration conf) { conf.add(Route(`/hello`, Text.fromPlain("Hello!"))) }

Or ensure BedSheet can find your AppModule. Do this by adding meta to your project's build.fan:

meta = [ ... "afIoc.module" : "myPod::AppModule", ... ]
